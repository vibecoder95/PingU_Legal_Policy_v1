# PingU Legal Documents

This folder contains the public legal documents for the PingU app, configured for hosting via GitHub Pages or any web hosting service.

## Files

- `privacy-policy.html` - Privacy Policy (Last Updated: January 2025)
- `terms-of-service.html` - Terms of Service (Last Updated: January 2025)
- `account-deletion.html` - Account & Data Deletion Policy (Last Updated: January 2025)

## Document Overview

### Privacy Policy
Comprehensive privacy policy covering:
- Data collection and usage
- Data security measures
- User rights including data deletion
- Clear distinction between chat deletion and account deletion
- Contact information for privacy inquiries

### Terms of Service
Legal terms covering:
- Age requirements (18+)
- User responsibilities
- Prohibited conduct
- Account termination policies
- Chat deletion vs account deletion clarification
- Contact information

### Account Deletion Policy
Detailed information about:
- How to delete a chat from your list (local removal)
- How to delete your account (server-side deletion)
- What data gets deleted and what doesn't
- Difference between chat deletion and account deletion
- Immediate and permanent deletion process

## Key Features

### Data Deletion Clarifications

**Chat Deletion (from list):**
- Removes chat from user's view only
- Chat remains on server if other participants exist
- Other users can still see the conversation
- Local removal, not server deletion

**Account Deletion:**
- Permanently deletes all user messages from server immediately
- Deletes all chat documents where user participated
- Removes account information and profile data
- Immediate, permanent, and irreversible

## GitHub Pages Setup

These files are configured to be served via GitHub Pages from the `/docs` folder.

### URLs

Once GitHub Pages is configured with `/docs` as the source:

- Privacy Policy: `https://[your-username].github.io/[repository-name]/privacy-policy.html`
- Terms of Service: `https://[your-username].github.io/[repository-name]/terms-of-service.html`
- Account Deletion: `https://[your-username].github.io/[repository-name]/account-deletion.html`

### Configuration Steps

1. Go to repository Settings → Pages
2. Set Source to: Branch `main` (or `master`) / Folder `/docs`
3. Save settings
4. Wait 1-2 minutes for deployment
5. Access your documents at the URLs above

### Example URLs

If your repository is `PingU_Legal_Policy` under username `vibecoder95`:

- Privacy Policy: `https://vibecoder95.github.io/PingU_Legal_Policy/privacy-policy.html`
- Terms of Service: `https://vibecoder95.github.io/PingU_Legal_Policy/terms-of-service.html`
- Account Deletion: `https://vibecoder95.github.io/PingU_Legal_Policy/account-deletion.html`

## Google Play Store Integration

These HTML files are designed to meet Google Play Store requirements:

✅ **Privacy Policy URL** - Required for all apps
✅ **Data Safety Compliance** - Covers data collection, usage, and deletion
✅ **User Rights Information** - GDPR and similar regulation compliance
✅ **Clear Deletion Policies** - Explains data deletion processes

### For Play Console Submission

1. Host these files publicly (GitHub Pages recommended)
2. Provide the Privacy Policy URL in Play Console
3. Reference these policies in your app's Data Safety form
4. Ensure URLs are accessible and mobile-friendly
